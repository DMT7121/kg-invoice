1

Import file CukCuk

Kéo thả hoặc chọn file hóa đơn từ CukCuk (.xlsx, .xls, .csv). Có thể chọn nhiều file cùng lúc.

2

Xem trước dữ liệu

Kiểm tra dữ liệu CukCuk đã được đọc đúng và xem trước kết quả chuyển đổi sang định dạng Sapo.

3

Xuất file Sapo

Tải file đã chuyển đổi về máy dưới dạng Excel hoặc CSV. Bạn có thể bổ sung thông tin khách hàng sau.

📋 Mapping dữ liệu:
  • Mã chứng từ gốc: Số hóa đơn CukCuk
  • Người mua hàng: Tên khách hàng (KH)
  • Số điện thoại: SĐT khách hàng
  • Hình thức thanh toán: Tự động nhận diện (Tiền mặt/Chuyển khoản/TM+CK)
  • Tên hàng hóa/DV: Tên món từ CukCuk
  • ĐVT, Số lượng, Đơn giá: Giữ nguyên từ CukCuk
  • Chiết khấu: Tiền KM từ CukCuk
  • Thuế GTGT: % VAT từ CukCuk
  • Các trường khác: Để trống để bổ sung sau
